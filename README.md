### 🚀 Featured Projects

#### 🧠 AI Dimag — Verified Memory for AI Coding Agents
**Side Project** | [Website](https://aidimag.com) | [npm](https://www.npmjs.com/package/aidimag) | [Cloud](https://cloud.aidimag.com)

Persistent memory system for MCP-compatible AI agents (Claude, Cursor, Copilot, Windsurf) that stores **falsifiable claims with grounding evidence**.

**Key Features:**
- 🛡️ **Human-Gated Capture** - Nothing enters memory without approval
- ✅ **Verification Lifecycle** - Evidence re-runs against current repo; stale beliefs get flagged
- 🔍 **Hybrid Semantic Recall** - FTS5 keyword + vector KNN (OpenAI/Ollama)
- 🚦 **Guardrails & Skills** - Behavioral rules enforced via `dim check`
- 👥 **Team Mode** - Local-first replicas with cross-machine verification
- 🔌 **MCP Integration** - Works with Claude, Cursor, and all MCP-compatible tools

**Tech Stack:** TypeScript, Node.js, SQLite, FTS5, Vector Search, MCP Protocol, Ollama, OpenAI API

**Impact:** Free for teams ≤10 users under Elastic License 2.0

```bash
npm install -g aidimag
dim init && dim bootstrap && dim review
```
